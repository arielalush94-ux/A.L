---
description: מילות קוד מותאמות אישית לעבודה מול מאגר גיטהאב (סנכרון וגיבוי)
---

המשתמש הגדיר 2 מילות קוד מיוחדות לעבודה מול GitHub, להלן ההוראות לביצוע כשהוא כותב אחת מהן:

1. כאשר המשתמש כותב **תסנכרן**:
המטרה היא להוריד את המידע העדכני מגיטהאב למחשב הנוכחי (למשוך שינויים).
**פעולות לביצוע:**

- יש להריץ את הפקודה: `git pull origin main`

1. כאשר המשתמש כותב **תגבה**:
המטרה היא להעלות את כל השינויים שבוצעו במחשב הנוכחי אל המאגר בגיטהאב, כדי לשמור אותם לפני מעבר למחשב אחר.
**פעולות לביצוע:**

- יש להריץ את הפקודות הבאות ברצף:
`git add .`
`git commit -m "Auto backup: גיבוי מסוף יום העבודה"` (או הודעת קומיט דומה שמתאימה לשינויים)
`git push origin main`
